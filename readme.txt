the qr-atendence system has both parts of taking attendence ,
1.qr-code scaning
2.rollcalling

i  used pop ups to add or input data in a system ,that can be dragged and positioned any where
 in a page ,to reduce on the number of pages that coud delay 
 the loading of the system.

 The qr code attendence system allows a lecturer to generate 
 a qr code for a particular lecture and then after
  students scans the code to update there attendence.


To run this project create a database with
a name school_db,then open the config folder and hit
the file named create_tables ,then type the directory 
path into your browser and press enter to create the 
tables in thee database...no password needed.
